# 墨衡 · 人生五维成长 App 设计说明

**日期**: 2026-07-14  
**状态**: 已确认，进入原型实现

## 定位
人生五维（个人 / 家庭 / 事业 / 团队 / 信念）动态平衡成长 App。  
阶段主攻制：同时仅 1 个主攻维；目标追踪优先；习惯与复盘为配套。

## 决策摘要
- 形态：手机为主、桌面为辅
- 首要痛点：目标追踪
- 目标模型：阶段主攻维（主攻 Top3 + 他维保底）
- 方案：五维主攻驾驶舱（方案 B）

## 信息架构
### 手机
1. 衡 — 主攻进度、Top3、今日习惯条  
2. 习 — 习惯提醒与打卡  
3. 复 — 周复盘、五维打分  
4. 我 — 主攻切换、信念一句话  

### 桌面
左：五维雷达 / 中：目标看板 / 右：复盘与提醒

## 产品规则
1. 主攻维唯一（默认：个人）
2. 主攻目标 ≤ 3；他维保底各 ≤ 1
3. 连续 2 周 < 4 分触发救火提醒
4. 信念轻量化澄清，不强填完整原则

## 视觉
墨色编辑风；石灰底 + 墨字 + 砚青主色；标题衬线 / 正文无衬线。

## 目标可编辑（已实现）
目标按模板存储指标参数，可新建/编辑/删除：
- 睡眠：入睡时间、每周天数、周数
- 运动：次数、分钟、周数
- 深度工作：分钟区间、天数、周数
- 陪伴 / 产出 / 自定义：均可自由改指标
标题由指标自动生成；修改睡眠时间会同步习惯提醒。

## 进度自动计算（周维度）

**主进度（进度条）= 本周完成度**
`本周进度 = min(100, round(本周打卡天数 / 每周目标 × 100))`

本周窗口：按目标 `startDate` 切周  
`第 N 周 = [start + (N-1)*7, start + N*7 - 1]`

**次要信息 = 全程累计**  
`全程 = 周期内打卡 / (每周目标 × 周数)`

卡片展示示例：`本周 1/2 · 50%` ｜ `W4/4 · 全程 3/8`
